#  GigStar
Gigs is an app that helps people find gigs. <br/>

This project was built with React.js and Apollo graphql client. <br/>
It is a fork of [](https://github.com/codinger41/gigs-mobile)


## :arrow_forward: How to Run App

1. cd to the repo
2. Run `yarn start`
